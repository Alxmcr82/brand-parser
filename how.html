Как это работает

Brand Parser — инструмент для быстрого сбора информации о брендах по URL

Алгоритм работы

1

Вы вводите URL сайта компании. Парсер обрезает адрес до домена и загружает главную страницу.

2

Если сайт блокирует обычный запрос (403, Cloudflare, JS-рендеринг) — автоматически подключается браузер Playwright.

3

Из HTML извлекается описание бренда: сначала из meta-тегов, затем из текста страницы, в крайнем случае — из тега title.

4

По регулярным выражениям находятся все ссылки на социальные сети. Дубликаты удаляются, служебные URL (API, трекеры) фильтруются.

5

Для каждой найденной соцсети парсер запрашивает количество подписчиков через API платформы или со страницы профиля.

Режим AI

При включении тогла AI парсер отправляет HTML страницы в нейросеть (DeepSeek через OpenRouter), которая извлекает описание и ссылки. Regex-парсинг используется как запасной вариант, если AI недоступен. По умолчанию AI выключен.

Платформы с подписчиками

Для этих платформ парсер автоматически подтягивает количество подписчиков:

  • Telegram — со страницы t.me (+ определение бота)
  • VK — через VK API
  • YouTube — через YouTube Data API
  • Max — со страницы max.ru (+ определение бота)
  • Dzen — через Dzen API
  • OK — с мобильной версии m.ok.ru
  • Rutube — со страницы канала
  • VC.ru — со страницы профиля

Платформы без подписчиков

Ссылки определяются, но подписчики не доступны через открытые API:

Instagram Facebook Twitter / X LinkedIn TikTok Pinterest WhatsApp Behance Dribbble GitHub HeadHunter

Экспорт

Результаты сохраняются в историю браузера. Можно экспортировать в CSV или JSON — как отдельный результат, так и всю историю.
